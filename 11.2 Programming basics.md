# 11.2 Programming basics

## A first program
```c
int main(void) { // A program begins executing statements in main().
  int wage; // The int wage statement creates an integer variable named wage.

  wage = 20; // The wage = 20 statement assigns wage with 20.

  printf("Salary is "); // The printf statement outputs 'Salary is ' to the screen.
  printf("%d", wage * 40 * 52); // This printf statement outputs the result of wage * 40 * 52.
  printf("\n"); // The printf statement with "\n" moves cursor to next line.

  return 0; // 'return 0' statement ends the program.
}
```
