# 11.2 Programming basics

## A first program
```c
int main(void) { // A program begins executing statements in main().
  int wage; // Each statement typically ends with a semicolon.

  wage = 20;

  printf("Salary is ");
  printf("%d", wage * 40 * 52);
  printf("\n");

  return 0; // 'return 0' statement ends the program.
}
```
